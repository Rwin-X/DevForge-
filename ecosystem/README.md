# 🧬 Ecosystem Simulation (CLI)

> A lightweight command-line ecosystem simulation written in Python for learning and experimenting with autonomous creature behavior.

This project is **not a game**. Instead, it serves as a simple simulation environment where autonomous creatures exist inside a generated world and interact with basic environmental elements.

The project is designed as a foundation for experimenting with artificial life, object-oriented programming, and simulation architecture.

---

## 📖 Overview

The simulation creates a virtual ecosystem consisting of:

- A 2D map
- Autonomous creatures
- Food resources
- Water sources
- Living areas

Each creature acts independently and follows simple rules while interacting with its surroundings.

The simulation focuses on demonstrating how multiple entities can coexist and operate within a shared environment.

---

## 🌍 World

The environment represents a small ecosystem where different objects occupy positions on the map.

Examples include:

- Empty terrain
- Food
- Water
- Obstacles
- Creatures

The world updates continuously as the simulation progresses.

---

## 🐾 Creatures

Every creature exists as an independent object with its own state.

A creature may contain information such as:

- Position
- Energy
- Hunger
- Direction
- Current status

Creatures perform simple actions automatically according to predefined rules.

---

## 🍃 Simulation

During execution, the ecosystem continuously updates.

Typical events include:

- Moving around the map
- Searching for food
- Consuming resources
- Updating creature state
- Refreshing the environment

The project demonstrates how autonomous agents interact with a changing world over time.

---

## 🏗 Project Purpose

The primary goal of this project is educational.

It can be used to explore concepts such as:

- Object-Oriented Programming
- Simulation design
- Artificial life concepts
- Autonomous agents
- State management
- Grid-based environments
- CLI application architecture

---

## ⚙️ Technologies

## 🎯 Design Philosophy

The project emphasizes:

- Simplicity
- Readable code
- Modular architecture
- Easy extensibility
- Learning through experimentation

Rather than implementing complex AI, the simulation provides a clean starting point that can gradually evolve into more sophisticated systems.
