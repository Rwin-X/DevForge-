### `// about`

**devforge** is an archive of tools built through **vibe coding** — fast, exploratory sessions where an idea goes straight from thought to working code, without getting stuck in perfect architecture or endless polishing.

Each project here is its own self-contained run: one problem, one focused session, one output.
