# AI Resume → HTML Portfolio Generator

Create a beautiful, responsive personal portfolio website from resume information in minutes.

This project provides a modern web interface where you enter your professional information, skills, experience, education, certifications, and projects. Missing sections can be automatically enhanced with AI, producing a complete portfolio website as a single HTML file.

---

* Modern cyber-inspired interface
* Resume-to-Portfolio workflow
* AI-generated professional summaries
* AI-enhanced project descriptions
* AI-enhanced work experience
* Live portfolio preview
* HTML source viewer
* One-click HTML download
* Copy generated HTML to clipboard
* Responsive design
* Avatar upload support
* Dynamic sections

  * Experience
  * Education
  * Projects
  * Certifications
* Skill tags with quick input

## How It Works

1. Fill in your resume information.
2. Add your skills.
3. Leave optional fields blank if you want AI to generate them.
4. Click **Generate Portfolio**.
5. Preview the generated website instantly.
6. Copy or download the generated HTML file.

---

## Output

The generator creates a standalone HTML portfolio that includes:

* Responsive layout
* Animated UI
* Professional landing page
* Skills visualization
* Timeline experience section
* Project showcase
* Education section
* Certification section
* Contact links

No additional files are required for the generated portfolio.
